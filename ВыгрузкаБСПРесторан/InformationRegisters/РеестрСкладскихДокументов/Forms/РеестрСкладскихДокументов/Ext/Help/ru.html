Реестр складских документов предназначен для примера реализации нестандартного журнала документов.

В подсистеме Управление доступом журнал документов показывает пример использования вида доступа ПравоЧтенияПоИдентификатору и шаблона ограничения доступа #ПоЗначениям.

  • Список ограничивается по видам доступа, указанных для документов журнала: Оприходование товаров, Перемещение товаров, Списание товаров. При этом ограничение документов в журнале не точно совпадает с ограничением документов в их списках, что возможно путем создания 2-х записей для документов Перемещение товаров.

В подсистеме Подключаемые команды журнал документов показывает пример вывода команд для списков, расположенных на отдельных закладках, а также пример скрытия команд, не подходящих по типам к списку выделенных документов.

В подсистеме Варианты отчетов журнал документов показывает пример вывода команды контекстного отчета, расположенной в командной панели второстепенного списка.
